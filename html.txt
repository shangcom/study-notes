rel="stylesheet"는 HTML 문서에서 <link> 태그의 rel(relationship) 속성으로, 현재 문서와 연결된 외부 리소스의 관계를 정의합니다.
**rel="stylesheet"**는 해당 링크가 **스타일시트(CSS)**임을 명시하며, 브라우저에게 이 파일이 CSS 스타일 정보를 포함하고 있음을 알립니다.
<link href="styles.css" rel="stylesheet">
이 코드는 "styles.css"라는 파일을 HTML 문서의 스타일시트로 참조하고, 이 파일에 정의된 CSS 규칙을 페이지에 적용합니다.
