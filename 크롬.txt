개발자 도구

1. 네트워크

이름 :
요청한 url

헤더 (Headers):
요청과 응답에 사용된 HTTP 헤더 정보를 표시합니다.
요청 헤더(Request Headers)와 응답 헤더(Response Headers)를 각각 확인할 수 있습니다.
요청 URL, 메소드, 상태 코드, Content-Type 등의 정보를 포함합니다.

페이로드 (Payload):
HTTP 요청의 본문 내용을 표시합니다.
주로 POST, PUT 요청에서 서버로 전송된 데이터를 확인할 수 있습니다.

미리보기 (Preview):
응답 본문을 미리보기 형태로 표시합니다.
JSON, HTML, XML 등의 형식을 쉽게 확인할 수 있도록 트리 구조로 표시합니다.

응답 (Response):
서버로부터 받은 HTTP 응답 본문을 원본 그대로 표시합니다.
주로 텍스트, HTML, JSON 등으로 응답 데이터를 확인할 수 있습니다.

시작점 (Initiator):
요청이 발생한 원인이나 출처를 추적할 수 있습니다.
어떤 코드나 리소스가 해당 요청을 발생시켰는지 확인할 수 있습니다.

타이밍 (Timing):
요청과 응답의 시간 측정 정보를 제공합니다.
DNS 조회, 연결 시간, 응답 시간 등의 네트워크 성능을 분석할 수 있습니다.
